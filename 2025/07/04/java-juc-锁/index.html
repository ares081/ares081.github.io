java锁实现

Java中的锁是多线程编程的核心机制，用于控制对共享资源的并发访问，确保线程安全（Thread Safety）和数据一致性。主要有两种实现方式：内置锁（synchronized 关键字） 和 显式锁（java.util.concurrent.locks.Lock 接口及其实现类）。
